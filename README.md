# busccR

Welcome to the **Baylor Statistical Science Consulting Center (BUSCC)**.

The BUSCC supports researchers, students, and faculty across Baylor University by providing high-quality statistical consulting, methodological guidance, and data-driven collaboration. As a consultant, you play an essential role in helping clients translate research questions into rigorous, reproducible analyses.

The **busccR** package was developed to support consulting workflows within the center by providing tools, templates, and tracking for consulting projects.

---

## Purpose of this Package

The goal of **busccR** is to help BUSCC consultants:

- Maintain consistent consulting practices across projects  
- Access reusable functions and utilities for analysis and reporting  
- Improve efficiency in client-facing deliverables  
- Improve project worklfow and track project status 

This package is intended for internal use by BUSCC consultants and trainees.
